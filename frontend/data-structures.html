1. Arrays

Think of an array like a row of lockers in school. Each locker has a number (index) and can store one item. You can quickly access any locker if you know its number.

What is an Array?

  • Collection of elements of same type
  • Fixed size once created
  • Elements stored in contiguous memory
  • Fast access using index (O(1) time complexity)

When to Use Arrays?

  • When you know the size beforehand
  • When you need fast random access
  • When storing homogeneous data
  • Don't use if size changes frequently

Real-Life Example:
Days of week = ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
You can directly access Friday using index 4 (since it's zero-indexed).

Java 
// Declaration and initialization
int[] numbers = new int[5];  // Creates array of size 5
int[] scores = {95, 87, 92, 88, 91};  // Initialize with values

// Accessing elements
System.out.println(scores[0]);  // Output: 95

// Modifying elements
scores[2] = 100;  // Changes 92 to 100

// Array length
System.out.println(scores.length);  // Output: 5
Java 
public class ArrayTraversal {
    public static void main(String[] args) {
        int[] arr = {10, 20, 30, 40, 50};
        
        // Using for loop
        System.out.println("Using for loop:");
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
        
        // Using enhanced for loop
        System.out.println("\nUsing enhanced for:");
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}

Common Array Operations:

  • Insertion - O(n) at specific position (requires shifting elements)
  • Deletion - O(n) (requires shifting elements)
  • Search - O(n) linear search, O(log n) binary search (if array is sorted)
  • Access - O(1) instantaneous access via index

2. Linked Lists

Imagine a treasure hunt where each clue points to the next location. That's a linked list! Each node holds data and a 'pointer' to the next node.
HEAD â†’ [Data|Next] â†’ [Data|Next] â†’ [Data|Next] â†’ NULL Example: HEAD â†’ [10|â€¢] â†’ [20|â€¢] â†’ [30|NULL]

Types of Linked Lists:

  • Singly Linked List - Traverse in one direction only.
  • Doubly Linked List - Connects forward and backward.
  • Circular Linked List - Last node connects back to the first.

When to Use?

  • Frequent insertions and deletions without shifting.
  • When the total size is unknown or highly variable.
  • When you don't need fast random access via an index.
  • Don't use if you need fast access to middle elements.
Java 
class Node {
    int data;
    Node next;
    
    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class LinkedList {
    Node head;
    
    // Insert at beginning
    void insertAtBeginning(int data) {
        Node newNode = new Node(data);
        newNode.next = head;
        head = newNode;
    }
    
    // Insert at end
    void insertAtEnd(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = newNode;
            return;
        }
        Node temp = head;
        while (temp.next != null) {
            temp = temp.next;
        }
        temp.next = newNode;
    }
    
    // Display list
    void display() {
        Node temp = head;
        while (temp != null) {
            System.out.print(temp.data + " â†’ ");
            temp = temp.next;
        }
        System.out.println("NULL");
    }
    
    public static void main(String[] args) {
        LinkedList list = new LinkedList();
        list.insertAtEnd(10);
        list.insertAtEnd(20);
        list.insertAtBeginning(5);
        list.display();  // Output: 5 â†’ 10 â†’ 20 â†’ NULL
    }
}

3. Stacks

Stack of plates in a cafeteria. You can only add or remove from the top. Last plate added is the first one removed (LIFO - Last In First Out).

Real Examples:

  • Undo feature in text editors
  • Browser back button history
  • Function call stack during program execution
Java 
import java.util.Stack;

public class StackDemo {
    public static void main(String[] args) {
        Stack<Integer> stack = new Stack<>();
        
        // Push elements
        stack.push(10);
        stack.push(20);
        stack.push(30);
        System.out.println("Stack: " + stack);  // [10, 20, 30]
        
        // Peek - view top
        System.out.println("Top element: " + stack.peek());  // 30
        
        // Pop - remove top
        int removed = stack.pop();
        System.out.println("Removed: " + removed);  // 30
        System.out.println("After pop: " + stack);  // [10, 20]
        
        // Check if empty
        System.out.println("Is empty? " + stack.isEmpty());  // false
    }
}

4. Queues

Like a queue at a ticket counter. The first person in line gets served first (FIFO - First In First Out).

Common Operations:

  • enqueue(x) / offer(x) - Add element to the rear
  • dequeue() / poll() - Remove element from the front
  • peek() - View the front element
Java 
import java.util.Queue;
import java.util.LinkedList;

public class QueueDemo {
    public static void main(String[] args) {
        Queue<Integer> queue = new LinkedList<>();
        
        // Add elements
        queue.offer(10);
        queue.offer(20);
        queue.offer(30);
        System.out.println("Queue: " + queue);  // [10, 20, 30]
        
        // Peek front
        System.out.println("Front: " + queue.peek());  // 10
        
        // Remove front
        int removed = queue.poll();
        System.out.println("Removed: " + removed);  // 10
        System.out.println("After removal: " + queue);  // [20, 30]
    }
}

5. Trees

A family tree! You have ancestors (parents) and descendants (children). A Binary tree means each parent has a maximum of 2 children left and right.
10 / \ 5 20 / \ / 3 7 15
Java 
class TreeNode {
    int data;
    TreeNode left, right;
    
    TreeNode(int data) {
        this.data = data;
        left = right = null;
    }
}

class BinaryTree {
    TreeNode root;
    
    // Inorder traversal (Left-Root-Right)
    void inorder(TreeNode node) {
        if (node == null) return;
        inorder(node.left);
        System.out.print(node.data + " ");
        inorder(node.right);
    }
    
    // Preorder traversal (Root-Left-Right)
    void preorder(TreeNode node) {
        if (node == null) return;
        System.out.print(node.data + " ");
        preorder(node.left);
        preorder(node.right);
    }
    
    // Postorder traversal (Left-Right-Root)
    void postorder(TreeNode node) {
        if (node == null) return;
        postorder(node.left);
        postorder(node.right);
        System.out.print(node.data + " ");
    }
    
    public static void main(String[] args) {
        BinaryTree tree = new BinaryTree();
        tree.root = new TreeNode(10);
        tree.root.left = new TreeNode(5);
        tree.root.right = new TreeNode(20);
        tree.root.left.left = new TreeNode(3);
        tree.root.left.right = new TreeNode(7);
        
        System.out.print("Inorder: ");
        tree.inorder(tree.root);  // 3 5 7 10 20
        
        System.out.print("\nPreorder: ");
        tree.preorder(tree.root);  // 10 5 3 7 20
        
        System.out.print("\nPostorder: ");
        tree.postorder(tree.root);  // 3 7 5 20 10
    }
}

6. Graphs

Like a social network! People are nodes, friendships are edges. You can traverse from one person to another through connections.
Adjacency List: 0 â†’ [1, 2] 1 â†’ [0, 3] 2 â†’ [0, 4] 3 â†’ [1] 4 â†’ [2]
Java 
import java.util.*;

class Graph {
    int V;  // Number of vertices
    LinkedList<Integer>[] adj;
    
    @SuppressWarnings("unchecked")
    Graph(int v) {
        V = v;
        adj = new LinkedList[v];
        for (int i = 0; i < v; i++) {
            adj[i] = new LinkedList<>();
        }
    }
    
    void addEdge(int v, int w) {
        adj[v].add(w);
        adj[w].add(v);  // For undirected graph
    }
    
    void BFS(int start) {
        boolean[] visited = new boolean[V];
        Queue<Integer> queue = new LinkedList<>();
        
        visited[start] = true;
        queue.add(start);
        
        System.out.print("BFS traversal: ");
        while (!queue.isEmpty()) {
            int node = queue.poll();
            System.out.print(node + " ");
            
            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.add(neighbor);
                }
            }
        }
    }
    
    public static void main(String[] args) {
        Graph g = new Graph(5);
        g.addEdge(0, 1);
        g.addEdge(0, 2);
        g.addEdge(1, 3);
        g.addEdge(2, 4);
        g.BFS(0);  // Output: 0 1 2 3 4
    }
}
